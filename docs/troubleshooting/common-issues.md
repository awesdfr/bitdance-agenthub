# Common Issues

Purpose: map user-visible symptoms to likely causes and recovery actions.

Scope:
- Stuck thinking, repeated actions, wrong completion claims, browser failures, Skill install failures, and approval friction.

| Symptom | Likely cause | Recovery action |
| --- | --- | --- |
| 一直思考中 | API超时 | 检查连接 |
| 反复做一件事 | 循环 | 查看计划手动调整 |
| 说完成但不对 | 误解任务 | 更多上下文 |
| 浏览器失败 | 页面结构变 | 更新策略 |
| Skill安装失败 | 依赖缺失 | 手动安装 |
| 内存高 | 上下文大 | 启用压缩 |
| 跑得慢 | 模型响应慢 | 换模型 |
| 创建后不能跑 | 试用期 | 转正 |
| Canvas红色 | 上游失败 | 查错误日志 |
| 审批多 | 自主低 | 提升级别 |
| 密钥错误 | 配置问题 | 检查Vault和Scope |
